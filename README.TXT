Send_mail - программа позволяющая организовать рассылку электронных писем на основании списка адресов со статистическими HTML шаблононами(в том числе содержащие графические файлы.)

Требуемые действия для отправки:
1) Требуется разместить файл(ы) EXCEL(только формат xlsx, xlsm) в каталоге (/in) c содержащими e-mail адресами(далее адреса) на >>>1 листе<<<(наименование не важно);
2) Указать в параметрах номера столбцов где содержутся адреса(цельночислительное значение). При этом желательно чтобы входящие .xlsx файлы были одинаковой структуры
   Допускается указание через запятую несколько столбцов для специфических списков где требуется отправить только на один валидный  e-mail строки(который
   расположен ранее в этом перечислении.)
3) Если требуется провести первичную отправку сообщений, то необходимо удалить файл exclude_email.csv(по-умолчанию, меняется в файле settings.ini)
    (в файле в столбик записываются адреса на которые ранее уже были высланны сообщения),
    т.к. он может содержать адреса встречающихся во вновь поступивших списках. и/или например ранее уже был отправлено это письмо
    но требуется исключить те адреса которые ранее уже получали данный тип письма;
4) Создать файл шаблон HTML письма для отправки сообщения(по-умолчанию pattern\pattern_html.html(меняется в файле settings.ini)).
    4.1 Для отправки HTML с содержащимимся графическим(и) изображениями требуется:
        - прописать в settings.ini(параметр: files_image_for_include) путь от корневой папки программы до файла изображений через запятую(
            например: "files_image_for_include=img\logo.png,img\other_picture.png" и т.д. через запятую без кавычек.);
        - прописать в шаблоне html в требуемые места размещения приамбулу указанной в файле settings.ini параметре id_image_for_include (по-умолчанию include)
            и номер файла в очереди указанном в параметре files_image_for_include(например src="cid:include0" где cid: - обязательяная константа, include - имя вставки из параметра,
            0(ноль) позиция файла в очереди(отсчет с нуля!) указанном в параметре files_image_for_include. В данном примере в месте размещения
            тега img с атрибутом src="cid:include0" будет размещен файл "img\logo.png" c примером из предыдущего пункта меню.

Рекомендуется перед отправкой сообщения создать тестовый xlsx файл с адресами различных популярных серверов
для тестирования отправки и отображения содержания на разных клиентах.
